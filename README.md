## setting

Đầu tiên, tải repository về máy tính:

```bash
$ git clone 
```

Cài đặt requirements:

```bash
$ cd custom_model_manager
$ pip install -r requirements.txt
```

Tạo database:

```bash
$ python manage.py migrate
```
Chạy development server

```bash
$ python manage.py runserver
```
